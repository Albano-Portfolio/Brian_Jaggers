# Brian Jaggers Resume Website

This is a static GitHub Pages-ready resume/CV website.

## Files included

- `index.html`
- `styles.css`
- `script.js`
- `README.md`

No React, no npm, no Vercel, and no build step are required.

## How to upload to GitHub

1. Create a new GitHub repository.
2. Upload these four files to the root/main folder of the repo:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
3. Go to **Settings**.
4. Go to **Pages**.
5. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
6. Click **Save**.
7. Wait a few minutes for GitHub Pages to publish the website.

## Edit contact info

Open `index.html` and update the phone number or email if needed.
